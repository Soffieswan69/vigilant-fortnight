Contributor profile

Jun 26, 2019
Michael Friesenhahn
  • GitHub

Add yourself to the contributors list #

  1. Add a new object to contributors.json with the following structure. Make sure to choose a unique contributor slug.

    "contributorslug": {
      "name": {
        "given": "Jaimie",
        "family": "Smith"
      }
    },

  2. If you want links to your online accounts to appear in your contributor lockup, add any of these members to your contributor object.

    "jaimiesmith": {
      "name": {
        "given": "Jaimie",
        "family": "Smith"
      },
      "twitter": "jaimiesmith",
      "github": "jaimiesmith",
      "glitch": "jaimiesmith"
    },

  3. You can also add supplemental info, but it's not currently displayed on web.dev.

    "jaimiesmith": {
      "name": {
        "given": "Jaimie",
        "family": "Smith"
      },
      "twitter": "jaimiesmith",
      "github": "jaimiesmith",
      "glitch": "jaimiesmith",
      "org": {
        "name": "Google",
        "unit": "Developer Relations"
      },
      "country": "US",
      "homepage": "https://jaimiesmithis.cool/",
      "description": {
        "en": "Jaimie is more than meets the eye"
      }
    },

Create a profile image #

  1. Select a photo of yourself that clearly shows your face and upper shoulders.
  2. Save two versions of the photo in /images/authors:
    • yourauthorslug.jpg (96 px x 96 px)
    • yourauthorslug@2x.jpg (192 px x 192 px)
