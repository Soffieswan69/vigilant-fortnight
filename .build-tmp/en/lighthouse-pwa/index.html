PWA audits

Measure if your site is fast, reliable and installable.

Overview

These checks validate the aspects of a Progressive Web App.

What you'll learn

  • Fast and reliable
  • Installable
  • PWA optimized
  • Additional items to manually check

Fast and reliable #

  • Page load is fast enough on mobile
  • Current page responds with a 200 when offline
  • Start URL responds with a 200 when offline

Installable #

  • Uses HTTPS
  • Registers a service worker that controls page and start_url
  • User can be prompted to install the web app

PWA optimized #

  • Avoid multiple page redirects
  • Configured for a custom splash screen
  • Sets an address-bar theme color
  • Content is not sized correctly for the viewport
  • Has a viewport tag with width or initial-scale
  • Contains some content when JavaScript is not available

Additional items to manually check #

  • Site works cross-browser
  • Page transitions don't feel like they block on the network
  • Each page has a URL
